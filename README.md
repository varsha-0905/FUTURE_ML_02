# Support Ticket Classification

## Objective

Build an NLP-based system to automatically classify customer support tickets and analyze ticket priorities.

## Tools Used

* Python
* Pandas
* NLTK
* Scikit-learn
* Matplotlib

## Features

* Text cleaning
* Stopword removal
* Tokenization
* TF-IDF vectorization
* Ticket category classification
* Priority analysis
* Model evaluation
* Confusion matrix visualization

## Evaluation Results

* Accuracy: 0.19
* Precision: 0.18
* Recall: 0.19

## Business Value

This project helps businesses automate support operations, prioritize urgent issues, reduce manual effort, and improve customer satisfaction.
